Before uploading files

  1. Make sure, that your file structure looks as followed:

    A class folder contains folders with lecture slides as well as the 'details.csv'. If you want a logo to be displayed, place image in the root folder. Make sure to call it "logo.jpg"

  2. Include the 'details.csv' file in every lecture folder

  3. Make sure that 'detail.csv' looks as followed:

    The first line indicates lecture title. Every next line contains subchapter's title and the slide number, where this subchapter starts.

  4. If you include a link in the third column, it will be automatically added to the web page right after the subchapter title.:

  5. Please use only .jpg, .jpeg or .png for the slides. Other file formats will be ignored.

  6. Do NOT rename the compressed folder.

  7. You can use any names for the folders and slides, but it can affect their order - names are sorted alphabetically.
