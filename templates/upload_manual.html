Before uploading files

  1. Make sure, that your file structure looks as followed:

    A class folder contains folders with lecture slides as well as the 'details.csv'

  2. Include the 'details.csv' file in every lecture folder

  3. Make sure that 'detail.csv' looks as followed:

    The first line indicates lecture title. Every next line contains subchapter's title and the slide number, where this subchapter starts.

  4. Please use only .jpg, .jpeg or .png for the slides. Other file formats will be ignored.

  5. Do NOT rename the compressed folder.

  6. You can use any names for the folders and slides, but it can affect their order - names are sorted alphabetically.
